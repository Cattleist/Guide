La dernière étape de ce guide est l'installation d'arm9loaderhax et la mise en place de Luma3DS pour qu'il s'exécute juste quelques millisecondes après l'allumage. Ceci est effectué en utilisant SafeA9LHInstaller d'[AuroraWright](https://github.com/AuroraWright/).
{: .notice}

Nous allons installer la [version d'AuroraWright](https://github.com/AuroraWright/arm9loaderhax) de arm9loaderhax.
{: .notice--info}

Nous allons également configurer la possibilité de lancer des payloads à partir d'arm9loaderhax, ce qui vous permettra de débricker votre SysNAND dans des situations qui vous brickeraient en temps normal en la restaurant à partir d'une sauvegarde.
{: .notice--info}

**Vous ne devez en aucun cas utiliser l'OTP d'une console différente ou je vous GARANTIS que vous brickerez votre console.**
{: .notice--danger}

#### Vue d'ensemble

Dans cette section, nous allons faire ce à quoi toutes les autres étapes nous ont menés : installer arm9loaderhax.

Il s'agit quasiment du meilleur genre d'exploitation de votre console possible car c'est installable de manière permanente dans les partitions NAND firm et que ça se lance avant presque tout le système d'exploitation, ce qui lui permet non seulement de fonctionner sur *toutes* les versions une fois installé, mais aussi de se protéger et de se remettre de la plupart des situations qui causeraient un brick d'une 3DS n'étant pas sous A9LH, telle qu'un HOME menu défectueux ou la mauvaise installation d'une application.

Le fichier `arm9loaderhax.bin` est ce qui est lancé directement par arm9loaderhax après qu'il ait finit de se charger depuis la NAND, et il peut être n'importe quelle payload arm9 valide. Ce fichier peut être remplacé à tout moment, cependant Luma3DS permet de lancer d'autres payloads arm9 en appuyant sur un bouton au lancement.

Dans ce cas, nous utilisons Luma3DS de [AuroraWright](https://github.com/AuroraWright/) pour démarrer directement une SysNAND patchée, nous permettant de contourner complètement la nécessité d'une EmuNAND quelconque, ce qui simplifie grandement l'utilisation d'une 3DS piratée en plus d'économiser de l'espace sur votre carte SD.

Une fois qu'arm9loaderhax est installé et que Luma3DS est réglé avec les bons paramètres, nous allons restaurer la sauvegarde réalisée précédemment.

Durant cette étape, nous allons également configurer les programmes suivants :    

+  **FBI** *(Permet d'installer des jeux et applications au format CIA)*
+  **Luma3DS Updater** *(Permet de mettre à jour facilement notre CFW installé)*
+  **Hourglass9** *(Couteau-suisse  qui permet d'interagir avec la NAND ou les cartouches)*

### Instructions

##### Section I - Préparation

{% capture notice-5 %}
**Assurez-vous d'utiliser une carte SD non corrompue !**

**L'utilisation d'une carte SD corrompue sans la réparer peut causer un BRICK !**

**Si vous pensez que votre carte SD pourrait être corrompue, faites un scan avec [H2testw (Windows)](h2testw-(windows)), [F3 (Linux)](f3-(linux)), ou [F3X (Mac)](f3x-(mac)) !**
{% endcapture %}

<div class="notice--danger">{{ notice-5 | markdownify }}</div>

1. **Si il existe, copiez le répertoire `/files9/` depuis votre carte SD vers un emplacement sécurisé sur votre ordinateur et faites en plusieurs copies à plusieurs endroits (tels que des services de stockage en ligne); les fichiers qu'il contient pourraient vous sauver de la perte totales de vos données si vous cassez votre système**
2. Créez un répertoire nommé `cias` à la racine de votre carte SD s'il n'existe pas déjà
3. **Supprimez le répertoire `a9lh` à la racine de votre carte SD s'il existe**
  + **Si vous installez arm9loaderhax accidentellement en utilisant l'`OTP.bin` d'une autre console, cela causera un BRICK !**
4. Supprimez le répertoire `3ds` à la racine de votre carte SD s'il existe
5. **Copiez _le contenu du_ dossier `starter` de l'archive `starter.zip` à la racine de votre carte SD**
  + Cela va inclure un tout nouveau répertoire `3ds` pour remplacer celui que vous venez de supprimer
6. Copiez _le contenu du_ `.7z` de SafeA9LHInstaller à la racine de votre carte SD
7. **Copiez le répertoire `a9lh` du `.zip` de data_input à la racine de votre carte SD**
8. **Copiez _le contenu du_ `.7z` d'arm9loaderhax dans le répertoire `a9lh` sur votre carte SD**
9. Copiez le fichier `hblauncher_loader.cia` se trouvant dans le `.zip` hblauncher_loader vers le répertoire `/cias/` de votre carte SD
10. Copiez le fichier `lumaupdater.cia` se trouvant dans le `.zip` Luma3DS Updater vers le répertoire `/cias/` de votre carte SD
11. Copiez le fichier `FBI.cia` se trouvant dans le `.zip` FBI vers le répertoire `/cias/` de votre carte SD
12. **Copiez le fichier `arm9loaderhax.bin` se trouvant dans le `.7z` Luma3DS à la racine de votre carte SD en écrasant les fichiers existants**
13. Créez un répertoire appelé `luma` à la racine de votre carte SD
14. Créez un répertoire appelé `payloads` dans le répertoire `luma` sur votre carte SD
15. Copiez le fichier `Hourglass9.bin` se trouvant dans le `.zip` Hourglass9 vers le répertoire `/luma/payloads/` sur votre carte SD et renommez-le en `start_Hourglass9.bin`
16. Copiez le fichier `GodMode9.bin` se trouvant dans le `.zip` GodMode9 vers le répertoire `/luma/payloads/` sur votre carte SD et renommez-le en `up_GodMode9.bin`
17. Copiez `aeskeydb.bin` vers le répertoire `/files9/` sur votre carte SD
18. Copiez `DspDump.3dsx` vers le répertoire`/3ds/` sur votre carte SD
19. Copiez _le contenu de_ `fbi-2.4.2-injectable.zip` vers le répertoire `/files9/` sur votre carte SD

### Section II - Installation d’arm9loaderhax

1. Réinsérez votre carte SD dans la console
2. Suivez ces étapes pour installer arm9loaderhax sur votre console :
  + Vous devez être en version 2.1.0
  + Rendez-vous sur la page `http://dukesrg.github.io/2xrsa.html?arm11.bin` via le navigateur de votre 3DS
    + Si vous obtenez l'erreur "This service is not available in your region", réglez le pays de la console dans les paramètres de la console pour qu'il corresponde à la region de la NAND installée avec 2.1.0 ctrtransfer
    + Si vous rencontrez une erreur, [suivez ce guide de dépannage](troubleshooting#ts_browser)
    + Si vous obtenez un écran glitché, suivez [ce guide de dépannage](troubleshooting#ts_safe_a9lh_screen)
    + Si le Wifi de votre console est désactivé, débranchez la 3DS et retirez la batterie pendant plusieurs secondes pour le réactiver
  + Appuyer sur (Select) pour lancer l'installation
  L'installateur va maintenant installer arm9loaderhax sur votre console (cette opération est très rapide)
  Éteignez votre console, maintenez enfoncé le bouton Power jusqu'à ce que la console soit éteinte si nécessaire
  + Copiez le fichier `OTP.bin` spécifique à votre console se trouvant dans le dossier `/a9lh/` de votre carte SD vers un endroit sur et n'hésitez pas à en faire plusieurs copies (sur le cloud par exemple), puis réinsérez la carte SD dans votre 3DS

##### Section III - Configuration de Luma3DS

1. Maintenez la touche (Select) au démarrage pour entrer dans le menu Luma3DS
  + Assurez-vous de commencer à appuyer sur le bouton avant d'appuyer sur Power
  + Si vous obtenez un écran noir, [suivez ce guide de dépannage](troubleshooting#ts_sys_a9lhdown)   
  + Si SafeA9LHInstaller démarre (à la place du menu de Luma3DS), [suivez ce guide](troubleshooting#ts_safe_a9lh) dépannage
2. Utilisez le bouton (A) et la croix directionnelle pour activer les éléments suivants :    
  + **"Autoboot SysNAND"**
  + **"Use SysNAND FIRM if booting with R"**
  + **"Show NAND or user string in System Settings"**
3. Si vous utilisez une **New 3DS**, vous devriez **également** activer l'élément suivant:
  + **"New 3DS CPU" to "Clock+L2(x)"**
    + Cela augmentera le framerate (image par secondes) dans beaucoup de jeux, mais peut causer des instabilités dans d'autres
    + Si certains jeux ne fonctionnent pas correctement, désactivez cette option et retentez votre chance
4. Appuyez sur (Start) pour enregistrer les modification et redémarrer
  + Si vous obtenez un écran noir, continuez simplement vers la section suivante   
  + Si vous obtenez une erreur "Failed to mount CTRNAND", continuez simplement vers la section suivante     

##### Section IV - Restauration du système

Si, avant de suivre ce guide, vous aviez déjà installé une EmuNAND et que vous souhaitez transférer ces données vers votre SysNand CFW, veuillez suivre [ce guide](move-emunand) à la place des 4 premières étapes de cette section.
{: .notice--info}

1. Ouvrez Hourglass9 à partir d'arm9loaderhax en maintenant (Start) enfoncé au démarrage
2. Sélectionnez "SysNAND Backup/Restore"
3. Sélectionnez "SysNAND Restore (keep a9lh)" et restaurez à aprtir du fichier `NANDmin.bin`
4. Appuyez sur (Start) pour redémarrer
  + Si vous obtenez un écran noir, suivez les instructions du [ctrtransfer 9.2.0](9.2.0-ctrtransfer)
5. Si votre sauvegarde est une version comprise entre 3.0.0 et 4.5.0, votre console ne redémarrera pas tant que vous ne téléchargez pas le firmware requis:
  + Téléchargez [ce fichier](http://nus.cdn.c.shop.nintendowifi.net/ccs/download/0004013800000002/00000056) et renommez-le `firmware.bin`
  + Téléchargez [ce fichier](http://nus.cdn.c.shop.nintendowifi.net/ccs/download/0004013800000002/cetk)
  + Copiez les fichiers `firmware.bin` et `cetk` dans le dossier `/luma/` de votre carte SD
  + Effacez ces 2 fichiers après avoir mis à jour votre 3DS
2. Mettez à jour votre 3DS en allant dans les Paramètres de la console puis "Autres paramètres", puis tout à droite sélectionnez "Mise à jour du système"
  + Mettre à jour sa console une fois que A9LH et Luma sont installés (ce que vous avez maintenant) ne pose aucun problème, arrêtez de poser la question
  + L'avertissement interdisant de faire une mise à jour sur une New 3DS en 2.1.0 ne s'applique plus après avoir restauré la sauvegarde de votre NAND.
  + Si vous obtenez une erreur, réglez vos paramètres DNS sur "Automatique"
  + Si vous recevez toujours une erreur et que la version de votre NAND est inférieure à 9.2.0, veuillez suivre [ce guide](9.2.0-ctrtransfer)


##### Section V - Injection de FBI

1. Ouvrez Hourglass9 à partir d'arm9loaderhax en maintenant (Start) enfoncé au démarrage
2. Go to "SysNAND Backup/Restore", then select the "Health&Safety Dump" option to dump Health & Safety to `hs.app` **(you can use Up and Down / Left and Right to change the name)**
3. Press (B), then select the "Health&Safety Inject" option
4. Select the FBI injectable `.app` for your region
5. Press (A) and confirm to inject
6. Appuyez sur (Start) pour redémarrer
7. If you still launch to the stock Health & Safety app and have downgraded with Gateway in the past, [follow this troubleshooting guide](troubleshooting#gw_fbi)

##### Section VI - Finalizing setup

1. Open Health and Safety (which is now FBI)
2. Select "SD"
3. Select "cias"
4. Navigate to `FBI.cia` and press (A) to install
5. Navigate to `hblauncher_loader.cia` and press (A) to install
6. Navigate to `lumaupdater.cia` and press (A) to install
7. Return to the SD directory with (B)
8. Navigate to `arm9loaderhax.bin`, then press (A) on it and select the "Copy" option
9. Return to the FBI main menu with (B)
10. Select "CTR NAND"
11. Select "\<current directory>"
12. Select the "Paste" option, then press (A) to confirm
13. Exit with the home button
14. Launch the Homebrew Launcher from the home menu
15. Select "DSP Dump"
16. Press (Start) when prompted to exit
17. Reboot while holding Start to launch Hourglass9
18. Go to "SysNAND Backup/Restore", then select "Health&Safety Inject"
19. Select `hs.app` (the original one that doesn't contain FBI), then press (A) and confirm to inject
20. Press (Select) on the main menu to eject your SD card
21. Press (Start) to reboot with the SD card removed
  + Booting the device at least once with the SD card removed will allow you to configure the CTRNAND based luma installation
22. Utilisez le bouton (A) et la croix directionnelle pour activer les éléments suivants :    
  + **"Show NAND or user string in System Settings"**
23. Si vous utilisez une **New 3DS**, vous devriez **également** activer l'élément suivant:
  + **"New 3DS CPU" to "Clock+L2(x)"**
    + Cela augmentera le framerate (image par secondes) dans beaucoup de jeux, mais peut causer des instabilités dans d'autres
    + Si certains jeux ne fonctionnent pas correctement, désactivez cette option et retentez votre chance
24. Reinsert your SD card, then press Start to save and reboot!
