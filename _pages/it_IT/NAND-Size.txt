Un backup NAND valido dovrebbe avere una delle dimensioni riportate di seguito.
{: .notice--primary}

Occasionalmente potrai ottenere un backup di dimensioni diverse da quelle qui elencate. Non è un problema se il backup è più grande del dovuto, ma non deve essere mai *più piccolo* delle dimensioni qui riportate.
{: .notice--info}

Note that a *min size* NAND will always be the smallest possible size for the purpose of saving space, and may not match the size of other NAND backups from your device. This is fine, just make sure that it matches one of the sizes on this page.
{: .notice--info}

The brand name refers to the manufacturer of the NAND chip in your device, **not** your SD card.
{: .notice--warning}

**Il file di backup deve corrispondere ad una delle dimensioni riportate su questa pagina; se così non fosse, cancellalo e creane uno nuovo!**
{: .notice--danger}

## 2DS:    

    + NAND Toshiba:     
      988.807.168 bytes = 943 MiB    

    + NAND Toshiba:    
      1.979.711.488 bytes = 1.84 GiB    

    + NAND Samsung:    
      1.000.341.504 bytes = 954 MiB    

    + NAND Samsung:    
      1.300.234.240 bytes = 1.21 GiB    

## Old 3DS:    

    + NAND Toshiba:     
      988.807.168 bytes = 943 MiB    

    + NAND Samsung:    
      1.000.341.504 bytes = 954 MiB    

## New 3DS:    

    + NAND Toshiba:    
      1.979.711.488 bytes = 1.84 GiB    

    + NAND Samsung:    
      1.300.234.240 bytes = 1.21 GiB    

    + NAND Samsung:
      1.954.545.664 bytes = 1.82 GiB
