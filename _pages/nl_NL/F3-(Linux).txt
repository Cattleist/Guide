Deze pagina legt uit hoe je je SD-kaart checkt op fouten met F3.
{: .notice}

Afhankelijk van de grootte van je SD-kaart en de snelheid van je computer kan dit proces een aantal uur duren!
{: .notice--info}

Deze pagina is alleen voor Linux gebruikers. Als je geen Linux gebruikt, zie dan de [H2testw (windows)](h2testw-(windows)) of [F3X (mac)](f3x-(mac)) pagina.
{: .notice--info}

#### Benodigdheden

* De nieuwste versie van [F3](https://github.com/AltraMayor/f3/archive/v6.0.zip)

#### Instructies

1. Unzip het f3 `.zip` bestand
2. `cd` naar de bestandslocatie van f3
3. Voer `make` uit om F3 te compileren
4. Plaats je SD-kaart in je computer
5. Mount je SD kaart
6. Run `./f3write <your sd card mount point>`
7. Wait until the process is complete. Zie hieronder voor een voorbeeld van de uitkomst.

		$ ./f3write /media/michel/6135-3363/
		Free space: 29.71 GB
		Creating file 1.h2w ... OK!
		...
		Creating file 30.h2w ... OK!
		Free space: 0.00 Byte
		Average Writing speed: 4.90 MB/s

8. Voer `./f3read <jouw SD kaart mount locatie>` uit
9. Wait until the process is complete. Zie hieronder voor een voorbeeld van de uitkomst.

		$ ./f3read /media/michel/6135-3363/
		                  SECTORS      ok/corrupted/changed/overwritten
		Validating file 1.h2w ... 2097152/        0/      0/      0
		...
		Validating file 30.h2w ... 1491904/        0/      0/      0

		  Data OK: 29.71 GB (62309312 sectors)
		Data LOST: 0.00 Byte (0 sectors)
			       Corrupted: 0.00 Byte (0 sectors)
			Slightly changed: 0.00 Byte (0 sectors)
			     Overwritten: 0.00 Byte (0 sectors)
		Average Reading speed: 9.42 MB/s


If the test shows the result `Data LOST: 0.00 Byte (0 sectors)` your SD card is good and you can delete all `.h2w` files on the SD card
{: .notice--success}

If the test shows any other results, your SD card may be corrupted or damaged and you may have to replace it!
{: .notice--danger}
