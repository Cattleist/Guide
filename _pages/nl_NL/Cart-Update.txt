Alle spel cartridges bevatten een kopie van de benodigden systeem versie om dat spel te spelen, hiermee kan iedereen op een lagere versie een spel cartridge gebruiken om te updaten naar die versie (zelfs zonder internet connectie).
{: .notice--info}

Cart Updates will only deliver updates to core features, such as the System Settings, Home Menu, Nintendo 3DS Sound, etc. Cart Updates will not deliver updates to Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza or eShop.
{: .notice--info}

Dit betekent dat een cart-update NIET een browser kan installeren als je er nog geen hebt. This will **only** update the actual system version *(for example, 2.1.0-0 -> 6.0.0-0)*. The last number *(the -0)* will **not** change.
{: .notice--warning}

**Updating with a game cart containing the version 9.9.0 or above will remove your browser!**
{: .notice--danger}

#### Benodigdheden

* Een spel dat [de versie bevat] (http://www.3dsdb.com/) in de gespecificeerde grafiek

#### Instructies

1. Boot uw 3DS in 'recovery mode' door L + R + A + UP in te drukken terwijl hij opstart
2. Decline the update and reboot
3. Put the game in the system
4. Attempt to launch the game
5. Update when prompted
